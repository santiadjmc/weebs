# CiberKids - Aplicación Web Educativa de Ciberseguridad

## Descripción
CiberKids es una aplicación web interactiva diseñada para enseñar ciberseguridad a niños de forma divertida y educativa. La aplicación está completamente en español y utiliza elementos visuales atractivos, juegos interactivos y contenido educativo adaptado para niños.

## Características Principales

### 🎨 Diseño y UX/UI
- **Diseño Responsivo**: Optimizado para desktop, tablet y móvil
- **Tema Claro/Oscuro**: Toggle animado con transiciones suaves
- **Colores Kid-friendly**: Paleta de colores brillantes y atractivos
- **Tipografía Legible**: Fuente Fredoka optimizada para niños
- **Animaciones**: Microinteracciones y efectos hover

### 📚 Contenido Educativo
El contenido está organizado en 5 secciones principales:

1. **🔐 Creación de Contraseñas Fuertes**
   - Herramienta interactiva de verificación de fortaleza
   - Consejos y mejores prácticas
   - Juego constructor de contraseñas

2. **🎣 Prevención de Phishing**
   - Ejemplos de correos legítimos vs. phishing
   - Juego de detección de emails sospechosos
   - Señales de alerta a identificar

3. **📱 Hábitos Seguros en Redes Sociales**
   - Configuración de privacidad
   - Simulador de publicaciones
   - Reglas de oro para compartir contenido

4. **🛡️ Privacidad Online**
   - Protección de información personal
   - Quiz interactivo
   - Escenarios de la vida real

5. **💻 Uso Responsable de Dispositivos**
   - Cronómetro de uso saludable
   - Consejos de seguridad
   - Mejores prácticas de mantenimiento

### 🎮 Juegos Interactivos
- **Constructor de Contraseñas**: Crea contraseñas seguras paso a paso
- **Detective Anti-Phishing**: Identifica correos maliciosos
- **Guardián de la Privacidad**: Toma decisiones sobre privacidad

### ✨ Funcionalidades Técnicas
- **Navegación Suave**: Scroll animado entre secciones
- **Accesibilidad**: Cumple con estándares WCAG
- **Navegación por Teclado**: Soporte completo para usuarios con discapacidades
- **Lectores de Pantalla**: Etiquetas ARIA y texto alternativo
- **SEO Optimizado**: Meta tags y estructura semántica
- **Performance**: Carga rápida y optimizada

## Instalación y Uso

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- JavaScript habilitado
- Conexión a internet (para fuentes web)

### Instalación
1. Descargar o clonar el repositorio
2. Abrir `index.html` en un navegador web
3. ¡La aplicación estará lista para usar!

### Uso Recomendado
- **Edad objetivo**: 8-14 años
- **Tiempo de sesión**: 20-30 minutos
- **Supervisión**: Recomendada para niños menores de 10 años
- **Frecuencia**: Sesiones regulares para reforzar conceptos
